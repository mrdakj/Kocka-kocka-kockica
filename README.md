## :package: Installation
:exclamation: Requirements: Freeglut 

### Manual

1. Clone this repository somewhere on your machine.

    ```sh
    git clone https://github.com/MATF-RG17/RG001-kocka-kocka-kockica.git ~/

    ```
2. Compile

    ```sh
    cd RG001-kocka-kocka-kockica
	make

    ```

3. Start

    ```sh
	./main

    ```
